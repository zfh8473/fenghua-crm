### 视觉细节

**Header:**
- 背景：`#FFFFFF`
- 高度：64px
- 边框：底部 `1px solid #E5E7EB`
- Logo：左侧，24px 图标 + 文字
- 搜索框：浅灰背景 `#F9FAFB`，圆角 8px
- 用户头像：圆形，32px

**侧边栏:**
- 背景：`#F9FAFB`
- 宽度：240px
- 菜单项：高度 40px，圆角 8px
- 悬停：浅蓝背景 `#F0F9FF`
- 激活：靛蓝背景 `#6366F1`，白色文字

**工具栏:**
- 背景：白色
- 高度：56px
- 搜索框：圆角 8px
- 创建按钮：靛蓝背景 `#6366F1`，白色文字，圆角 8px

**表格 (核心设计):**
- 背景：白色
- 无边框设计
- 表头：浅灰背景 `#F3F4F6`，高度 48px，固定
- 行高：56px
- 行悬停：浅蓝背景 `#F0F9FF`（Airtable 特色）
- 文字：主文本 `#1F2937`，次要文本 `#6B7280`
- 单元格：可编辑（可选功能）

## 组件设计规范

### 表格组件（核心）

**表格容器:**
```
背景: #FFFFFF
圆角: 8px
无边框
表头固定
```

**表格行:**
```
行高: 56px
悬停: 背景 #F0F9FF (浅蓝)
边框: 无
内边距: 12px 16px
```

**表格单元格:**
```
可编辑: 点击编辑（可选）
对齐: 左对齐（文本），右对齐（数字）
字体: 14px
```

### 按钮组件

**主按钮:**
```

圆角: 8px
内边距: 12px 24px

```

## 实现代码示例

### Tailwind 配置
```typescript
// tailwind.config.ts
colors: {
  'airtable-bg': '#F9FAFB',
  'airtable-table-bg': '#FFFFFF',
  'airtable-header': '#F3F4F6',
  'airtable-hover': '#F0F9FF',
  'airtable-text': '#1F2937',
  'airtable-accent': '#6366F1',
}
```

### React 组件示例
```tsx
// Table Component
<table className="w-full border-collapse">
  <thead className="bg-[#F3F4F6] sticky top-0">
    <tr>
      <th className="p-4 text-left text-sm font-semibold text-gray-600">邮箱</th>
      <th className="p-4 text-left text-sm font-semibold text-gray-600">姓名</th>

      <td className="p-4 text-sm text-[#1F2937]">{name}</td>
    </tr>
  </tbody>
</table>
```
