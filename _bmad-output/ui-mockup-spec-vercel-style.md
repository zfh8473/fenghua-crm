### 视觉细节

**Header:**
- 背景：`#FFFFFF`
- 高度：64px
- 边框：底部 `1px solid #E5E7EB`（细线）
- Logo：左侧，24px 图标 + 文字，黑色
- 搜索框：白色背景，细线边框 `1px solid #E5E7EB`
- 用户头像：圆形，32px，细线边框

**侧边栏:**
- 背景：`#FAFAFA`
- 宽度：240px
- 菜单项：高度 40px，无圆角或小圆角 4px
- 悬停：浅灰背景 `#F5F5F5`
- 激活：黑色背景 `#000000`，白色文字

**工具栏:**
- 背景：`#FFFFFF`
- 高度：56px
- 边框：底部 `1px solid #E5E7EB`
- 搜索框：细线边框
- 创建按钮：黑色背景 `#000000`，白色文字，圆角 4px

**内容卡片:**
- 背景：`#FFFFFF`
- 边框：`1px solid #E5E7EB`（细线）
- 圆角：4px 或 0px（直角）
- 内边距：24px
- 悬停：边框颜色加深 `#D1D5DB`

**表格:**
- 边框：细线 `1px solid #E5E7EB`
- 表头：浅灰背景 `#FAFAFA`，高度 48px
- 行高：56px
- 行悬停：浅灰背景 `#F9FAFB`
- 文字：主文本 `#000000`，次要文本 `#666666`

## 组件设计规范

### 按钮组件

**主按钮 (Primary):**
```
背景: #000000 (黑色) 或 #0070F3 (Vercel 蓝)
文字: #FFFFFF
圆角: 4px
内边距: 12px 24px
字体: 14px, 500 weight
边框: 无
悬停: 背景 #0051CC (蓝色) 或 #333333 (黑色加深)
```

**次要按钮 (Secondary):**
