### 视觉细节

**Header:**
- 背景：`linear-gradient(180deg, #F6F9FC 0%, #FFFFFF 100%)`
- 高度：64px
- 边框：无
- Logo：左侧，24px 图标 + 文字，深蓝灰色 `#0A2540`
- 搜索框：白色背景，柔和阴影，圆角 12px
- 用户头像：圆形，32px，带柔和阴影

**侧边栏:**
- 背景：`linear-gradient(180deg, #F6F9FC 0%, #FFFFFF 100%)`
- 宽度：240px
- 菜单项：高度 44px，大圆角 12px
- 悬停：浅蓝灰背景 `#E8F0F8`
- 激活：紫色背景 `#635BFF`，白色文字

**工具栏:**
- 背景：白色，柔和阴影
- 高度：56px
- 圆角：16px
- 搜索框：圆角 12px，柔和边框
- 创建按钮：紫色渐变 `linear-gradient(135deg, #635BFF, #7C3AED)`

**内容卡片:**
- 背景：白色
- 圆角：20px（大圆角）
- 阴影：`0 2px 8px rgba(0, 0, 0, 0.08)`（精致阴影）
- 内边距：24px
- 悬停：阴影加深 `0 4px 16px rgba(0, 0, 0, 0.12)`

**表格:**
- 无边框设计
- 表头：浅蓝灰背景 `#F6F9FC`，高度 52px
- 行高：60px
- 行悬停：浅蓝灰背景 `#F0F7FF`
- 文字：主文本 `#0A2540`，次要文本 `#425466`

## 实现代码示例

### Tailwind 配置
```typescript
// tailwind.config.ts
colors: {
  'stripe-bg': '#F6F9FC',
  'stripe-card': '#FFFFFF',
  'stripe-text': '#0A2540',
  'stripe-text-secondary': '#425466',
  'stripe-accent': '#635BFF',
}
```

### React 组件示例
```tsx
// Card Component
<div className="bg-white rounded-[20px] shadow-[0_2px_8px_rgba(0,0,0,0.08)] p-6 hover:shadow-[0_4px_16px_rgba(0,0,0,0.12)] transition-shadow duration-300">
  {children}
</div>

// Button Component
<button className="bg-gradient-to-r from-[#635BFF] to-[#7C3AED] text-white px-7 py-3.5 rounded-xl font-medium shadow-[0_2px_8px_rgba(99,91,255,0.3)] hover:shadow-[0_4px_12px_rgba(99,91,255,0.4)] active:scale-[0.98] transition-all duration-200">
  创建新用户
</button>
```
