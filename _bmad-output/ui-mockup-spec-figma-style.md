### 视觉细节

**Header:**
- 背景：`#FFFFFF`
- 高度：64px
- 边框：底部 `2px solid #E5E7EB`
- Logo：左侧，多彩图标，24px
- 搜索框：浅灰背景 `#F7F7F7`，圆角 12px
- 用户头像：圆形，32px，彩色边框

**侧边栏:**
- 背景：`#FFFFFF`
- 宽度：240px
- 菜单项：高度 44px，大圆角 12px
- 悬停：浅色背景（根据菜单项颜色变化）
- 激活：彩色背景（蓝色 `#18A0FB`、紫色 `#7B68EE` 等），白色文字

**工具栏:**
- 背景：白色
- 高度：56px
- 搜索框：圆角 12px
- 创建按钮：蓝色背景 `#18A0FB`，白色文字，圆角 12px

**内容卡片:**
- 背景：白色
- 圆角：16px
- 边框：可选彩色边框（2px）
- 阴影：`0 2px 8px rgba(0, 0, 0, 0.08)`
- 内边距：24px
- 悬停：阴影加深，轻微上移

**表格:**
- 无边框设计
- 表头：浅灰背景 `#F7F7F7`，高度 52px
- 行高：60px
- 行悬停：彩色背景（根据角色颜色变化）
- 文字：主文本 `#000000`，次要文本 `#666666`

## 配色系统

### 主色调
- **蓝色**: `#18A0FB` - 主要操作
- **紫色**: `#7B68EE` - 次要操作
- **绿色**: `#00C853` - 成功状态
- **橙色**: `#FF6B35` - 警告/强调

### 角色标签颜色
- **管理员**: 蓝色 `#18A0FB`
- **总监**: 紫色 `#7B68EE`
- **前端专员**: 绿色 `#00C853`
- **后端专员**: 橙色 `#FF6B35`

## 实现代码示例

### Tailwind 配置
```typescript
// tailwind.config.ts
colors: {
  'figma-blue': '#18A0FB',
  'figma-purple': '#7B68EE',
  'figma-green': '#00C853',
  'figma-orange': '#FF6B35',
}
```

### React 组件示例
```tsx
// Card Component
<div className="bg-white rounded-2xl shadow-[0_2px_8px_rgba(0,0,0,0.08)] p-6 hover:shadow-[0_4px_16px_rgba(0,0,0,0.12)] hover:-translate-y-0.5 transition-all duration-300">
  {children}
</div>

// Button Component (蓝色)
<button className="bg-[#18A0FB] text-white px-7 py-3.5 rounded-xl font-medium hover:bg-[#0D8CE8] active:scale-[0.98] transition-all duration-200">
  创建新用户
</button>

// Role Badge (多彩)
<span className={`px-3 py-1.5 rounded-lg text-sm font-medium text-white ${
  role === 'ADMIN' ? 'bg-[#18A0FB]' :
  role === 'DIRECTOR' ? 'bg-[#7B68EE]' :
  role === 'FRONTEND_SPECIALIST' ? 'bg-[#00C853]' :
  'bg-[#FF6B35]'
}`}>
  {roleLabel}
</span>
```
