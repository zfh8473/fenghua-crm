# Twenty CRM 快速测试指南

**项目：** fenghua-crm  
**目的：** 快速测试 Twenty CRM 的核心功能

---

## 🚀 快速开始

### 1. 访问系统

打开浏览器访问：**http://localhost:3000**

### 2. 首次登录

- 如果是首次访问，系统会引导你创建账户
- 或者查看 Docker 日志获取默认凭据：
  ```bash
  cd ~/Documents/GitHub/twenty/packages/twenty-docker
  docker-compose logs server | grep -i "password\|admin\|login"
  ```

---

## 📋 核心功能测试清单

### 测试 1：客户信息管理（5-10 分钟）

**步骤：**

1. **创建客户**
   - 点击"Companies"或"客户"菜单
   - 点击"+"或"New"按钮
   - 填写客户信息：
     - 公司名称
     - 行业
     - 地址
     - 联系方式
   - 保存

2. **查看客户列表**
   - 检查列表显示
   - 测试搜索功能
   - 测试筛选功能
   - 测试排序功能

3. **编辑客户**
   - 点击客户进入详情页
   - 修改信息
   - 保存

4. **记录评估结果**
   - 在 `docs/twenty-evaluation-log.md` 中记录

---

### 测试 2：联系人管理（5-10 分钟）

**步骤：**

1. **创建联系人**
   - 找到"People"或"联系人"菜单
   - 创建新联系人
   - 填写联系人信息：
     - 姓名
     - 职位
     - 邮箱
     - 电话
   - 关联到客户

2. **测试关联关系**
   - 查看客户详情页的联系人列表
   - 查看联系人的客户关联

3. **记录评估结果**
   - 在评估日志中记录

---

### 测试 3：多语言支持（3-5 分钟）

**步骤：**

1. **查找语言设置**
   - 查看用户设置或系统设置
   - 查找语言切换选项

2. **测试语言切换**
   - 切换到中文（如果支持）
   - 检查界面翻译
   - 检查数据输入支持

3. **记录支持的语言**
   - 列出所有支持的语言
   - 评估中文支持程度

4. **记录评估结果**
   - 在评估日志中记录

---

### 测试 4：AI 客户分析（5-10 分钟）

**步骤：**

1. **查找 AI 功能**
   - 查看是否有 AI 分析菜单
   - 查看客户详情页是否有 AI 分析
   - 查看是否有智能推荐

2. **测试 AI 功能（如果有）**
   - 尝试使用 AI 分析
   - 查看分析结果
   - 评估功能实用性

3. **评估集成能力**
   - 查看 API 文档
   - 评估集成外部 AI 服务的可行性

4. **记录评估结果**
   - 在评估日志中记录

---

### 测试 5：GraphQL API（5-10 分钟）

**步骤：**

1. **访问 GraphQL Playground**
   - 打开：http://localhost:3000/graphql
   - 查看 API 文档

2. **测试查询**
   ```graphql
   query {
     companies {
       edges {
         node {
           id
           name
           domainName
         }
       }
     }
   }
   ```

3. **测试变更（Mutation）**
   - 尝试创建客户
   - 尝试更新客户

4. **记录评估结果**
   - 在评估日志中记录

## ⚠️ 常见问题

### 问题 1：无法访问前端

**解决方案：**
```bash
# 检查容器状态
cd ~/Documents/GitHub/twenty/packages/twenty-docker
docker-compose ps

# 查看日志
docker-compose logs server
```

### 问题 2：GraphQL API 无法访问

**解决方案：**
- 确保服务器容器正常运行
- 检查端口 3000 是否被占用
- 查看服务器日志

### 问题 3：功能找不到

**解决方案：**
- 查看 Twenty 官方文档
- 检查菜单和导航
- 查看用户指南
