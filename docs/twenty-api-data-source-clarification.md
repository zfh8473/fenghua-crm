# Twenty GraphQL API 数据来源说明

**项目：** fenghua-crm  
**目的：** 澄清 Twenty GraphQL API 的数据来源和作用  
**日期：** 2025-12-23

---

## ⚠️ 重要澄清

### Twenty GraphQL API 的作用

**关键理解：** Twenty CRM 是一个 **CRM 系统**，不是数据查询服务。

**Twenty GraphQL API 只能访问：**
- ✅ **你自己录入到 Twenty CRM 中的数据**
- ✅ **你自己公司的客户数据**
- ✅ **你自己导入的历史订单数据**

**Twenty GraphQL API 不能访问：**
- ❌ 全球其他公司的数据
- ❌ 外部数据库的历史订单
- ❌ 其他公司的客户信息
- ❌ 公开的市场数据

---

## 📊 数据来源说明

### 1. Twenty CRM 的数据来源

**数据需要你自己录入或导入：**

```
你的业务数据
    ↓
录入/导入到 Twenty CRM
    ↓
存储在 PostgreSQL 数据库
    ↓
通过 GraphQL API 访问
```

**数据来源方式：**

1. **手动录入**
   - 在 Twenty 界面中手动添加客户
   - 手动添加订单记录
   - 手动添加沟通记录

2. **批量导入**
   - 通过 CSV/Excel 导入客户数据
   - 导入历史订单数据
   - 导入联系人数据

3. **API 集成**
   - 通过 GraphQL Mutation 创建数据
   - 通过 REST API 同步数据
   - 通过 Webhook 接收数据

4. **第三方集成**
   - 集成邮件系统（自动同步邮件）
   - 集成日历系统（自动同步会议）
   - 集成其他业务系统

---

### 2. 历史订单数据的获取

**重要：** Twenty CRM 本身不提供历史订单查询功能。

**你需要：**

1. **从现有系统导出数据**
   - 从 Excel/CSV 文件导出
   - 从其他 CRM 系统导出
   - 从业务系统导出

2. **导入到 Twenty CRM**
   - 使用 Twenty 的导入功能
   - 或通过 GraphQL API 批量创建

3. **然后才能通过 API 查询**
   - 查询你自己导入的数据
   - 不能查询外部数据

---

## 🔍 如果需要外部数据

### 场景：查询全球公司的历史订单

**问题：** 如果你想查询某个潜在客户的历史订单（比如他们从其他供应商的订单），Twenty CRM 无法提供。

**解决方案：**

1. **第三方数据服务 API**
   - **海关数据 API**（如果客户是进出口公司）
   - **企业信息查询 API**（如：天眼查、企查查）
   - **贸易数据服务**（如：Trade Data API）

2. **集成流程：**
   ```
   第三方数据服务 API
       ↓ 获取外部数据
   你的集成服务
       ↓ 处理和存储
   Twenty CRM (通过 GraphQL API)
       ↓ 存储到 CRM
   可以通过 GraphQL 查询
   ```

## 💻 实际应用场景

### 场景 1：分析你自己的客户

**数据来源：** 你自己录入的客户数据

```
你的客户数据（已录入到 Twenty）
    ↓
GraphQL API 查询
    ↓
AI 分析服务
    ↓
生成客户画像
```

**这是 Twenty GraphQL API 的主要用途。**

---

### 场景 2：查询潜在客户的外部数据

**数据来源：** 第三方数据服务

```
第三方数据服务（如：海关数据 API）
    ↓
你的集成服务获取数据
    ↓
通过 GraphQL API 存储到 Twenty
    ↓
然后可以通过 GraphQL 查询
```

**Twenty 本身不提供外部数据，需要你自己集成。**

---

## 📋 总结

### Twenty GraphQL API 的真实能力

**可以做的：**
- ✅ 查询你自己录入的客户数据
- ✅ 查询你自己导入的订单数据
- ✅ 查询你自己的沟通记录
- ✅ 创建、更新、删除你自己的数据

**不能做的：**
- ❌ 查询全球其他公司的数据
- ❌ 查询外部数据库的历史订单
- ❌ 提供公开的市场数据
- ❌ 查询其他公司的客户信息

### 数据获取策略

**对于 AI 客户分析：**

1. **使用你自己的数据**（主要）
   - 从 Twenty CRM 获取你自己录入的数据
   - 这是 AI 分析的主要数据源

2. **集成外部数据**（可选增强）
   - 如果需要，集成第三方数据服务
   - 获取外部数据后存储到 Twenty
   - 然后用于分析

---

## 🎯 对进出口公司的建议

### 数据策略

**第一阶段：使用内部数据**
- 录入现有客户数据到 Twenty
- 导入历史订单数据
- 使用这些数据进行 AI 分析

**第二阶段：集成外部数据（可选）**
- 如果需要，集成海关数据 API
- 获取客户的进出口历史
- 存储到 Twenty 用于分析

**关键点：**
- Twenty 是数据存储和管理系统
- 数据需要你自己提供
- 外部数据需要你自己集成
